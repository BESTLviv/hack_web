Game

PC Game

Категорія, у якій перед учасниками стоїть завдання створити комп’ютерну гру для пк. Для цього вони можуть використовувати будь-які інструменти, засоби та методи. Єдиною умовою є те, щоб у грі були задіяні слова, які кожна команда отримає випадковим чином безпосередньо перед початком визаченого часу.

Mobile

Mobile application

У цій категорії учасники створюють додаток на смартфон. Випадковим чином їм надається галузь (наприклад транспорт, освіта, медицина і т.д.), яку вони повинні покращити своїм додатком. Тут на передній план виходить практичне застосування готової роботи, але й не потрібно забувати про гарний інтерфейс.

Web

Web-service

Категорія, де учасники створюватимуть онлайн-сервіс на одну з актуальних тем (наприклад е-голосування, хмарні технлогоії, чат-боти і т.д.). Одними з головних вимог до проектів буде їхня зручність та ефективність.
